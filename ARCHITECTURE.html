CVE-to-Program Matching

Runs after every feed poll. Scores each CVE against every active program's tech stack, CWE list, and scope. One CVE can match multiple programs — that's a feature (multiple payouts).

Auto-run Multi-match

Scoring Factors

Tech stack keyword overlap. High-value CWE match (XSS, SQLi, SSRF, RCE, auth bypass). CVSS severity weighting. Exploit availability bonus. CISA KEV flag bonus. Attack vector (NETWORK preferred).

0-100 Score

analyzeMatch() — Opus 4.6

High-scoring matches (≥70) get deep AI analysis: exploitation strategy, estimated payout, difficulty assessment, and recommended approach. Triggers the full bounty pipeline.
